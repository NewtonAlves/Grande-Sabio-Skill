# Uso Da Grande Sabio

A **Grande Sabio** deve ser usada quando o trabalho envolver criação, revisão, estrutura, continuidade ou diagnóstico de obra em português brasileiro.

Ela não foi criada para responder de forma genérica. A função dela é atuar com método: entender o pedido, escolher o modo certo, preservar a voz do autor e entregar uma solução editorial concreta.

## Pedidos Prontos

### Revisar Capítulo

```text
Use $grande-sabio para revisar este capítulo em PT-BR sem apagar minha voz.
```

### Criar Personagem

```text
Use $grande-sabio para criar uma protagonista juvenil com desejo, medo, falha ativa, contradição e voz própria.
```

### Diagnosticar Enredo

```text
Use $grande-sabio para analisar este enredo, apontar o que está genérico e propor correções concretas.
```

### Criar Mundo Ficcional

```text
Use $grande-sabio para criar uma cidade costeira com política, economia, religião, conflitos e consequências narrativas.
```

### Criar Lore De Jogo

```text
Use $grande-sabio para criar uma facção, três NPCs e uma quest curta para um RPG sombrio.
```

### Criar Pauta De Revista

```text
Use $grande-sabio para criar uma pauta de revista com título, subtítulo, lead, seções e pontos de apuração.
```

### Adaptar Para Audiolivro

```text
Use $grande-sabio para adaptar este trecho para leitura em voz alta, apontando pausas, fôlego e trechos difíceis de narrar.
```

### Continuar Projeto Longo

```text
Use $grande-sabio para continuar minha obra lendo primeiro project-memory.
```

## Modos Internos

| Modo | Quando usar |
| --- | --- |
| Escritor | cenas, capítulos, contos, diálogos e prosa |
| Redator | apresentações, sinopses, textos editoriais e revista |
| Roteirista | cenas, beats, viradas e estrutura dramática |
| Editor De Desenvolvimento | enredo, arco, ritmo e coerência |
| Editor Literário | voz, estilo, presença e força literária |
| Revisor PT-BR | ortografia, pontuação, concordância, regência e crase |
| Copidesque | clareza, fluidez, coesão e ritmo |
| Leitor Crítico | parecer honesto sobre qualidade narrativa |
| Criador De Personagens | desejo, medo, falha, voz e arco |
| Criador De Mundo | culturas, regiões, sistemas e consequências |
| Criador De Nomes | nomes, padrões fonéticos e banco de nomenclatura |
| Narrativa Para Jogos | lore, quests, NPCs, classes, itens e bestiário |
| Revista | pauta, lead, matéria, coluna, entrevista e editorial |
| Contos E Antologias | contos, microcontos, coletâneas e ordem temática |
| Memória E Continuidade | leitura e atualização de `project-memory/` |
| Auditoria Final | revisão final de obra, capítulo ou material editorial |

## Padrão De Entrega

Para criação, a skill entrega o texto primeiro quando o pedido for produtivo.

Para revisão, entrega:

- versão revisada;
- principais mudanças;
- pontos de atenção.

Para diagnóstico, entrega:

- problema;
- evidência;
- impacto;
- correção concreta.

Para continuidade, entrega:

- memória lida;
- decisão tomada;
- texto, diagnóstico ou revisão;
- atualização de memória quando necessário.

## Limites De Uso

A skill não deve:

- prometer publicação;
- prometer venda;
- prometer sucesso comercial;
- inventar fatos reais;
- inventar entrevistas;
- substituir advogado;
- substituir pesquisa técnica especializada;
- fazer tráfego pago, funis ou campanhas comerciais;
- usar português europeu como padrão.
